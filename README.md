# Conceptual Questions
## 1. What is an Abstract Data Type (ADT)?

An Abstract Data Type (ADT) is a way of defining a data structure based on what it does, not how it is implemented. It describes the operations that can be performed (like insert, delete, or search) without specifying the exact code or structure used to implement them.

## 2. Explain the difference between a queue (FIFO) and a stack (LIFO).

A queue follows FIFO (First In, First Out), meaning the first element added is the first one removed, like a line of people waiting.

A stack follows LIFO (Last In, First Out), meaning the last element added is the first one removed, like a stack of plates.

## 3. Name and briefly explain three types of data structures.
Examples of 3 Data structures:
1. Queue
2. Stack
3. Linked List

### Queue
A Queue is a First-in-first-out data structure (FIFO). It is a linear data Structure. How a FIFO structure works is that the first element addded is the first one taken out. Similar to lineups at fast food places. 

### Stack
A Stack is a Last-in-first-out data structure (LIFO). It is also considered a linear data structure. How the LIFO system works is that the first element added is the last to leave, and the last element added (which is the one at the top) is the first to leave.

### Linked List
A linked list is a type of data structure comprised of nodes and pointers arranged in a list. In linked lists a node represents 2 things the value and the memory address of the next node in the list.
