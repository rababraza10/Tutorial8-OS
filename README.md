Conceptual Questions
1. What is an Abstract Data Type (ADT)?

An Abstract Data Type (ADT) is a way of defining a data structure based on what it does, not how it is implemented. It describes the operations that can be performed (like insert, delete, or search) without specifying the exact code or structure used to implement them.

2. Explain the difference between a queue (FIFO) and a stack (LIFO).

A queue follows FIFO (First In, First Out), meaning the first element added is the first one removed, like a line of people waiting.

A stack follows LIFO (Last In, First Out), meaning the last element added is the first one removed, like a stack of plates.

4. Explain what a binary tree is, what are some common operations of a binary tree?

A binary tree is a data structure made up of nodes that can have at most 2 children; a left child and a right child. Each node holds data and the pointers to the children nodes. Common operations associeated with a binary tree are insertion: add a new node, Deletion: remove a node, Search: find a value, traversal: visit nodes in order, find height: measures how many nodes "tall" the tress it, and finally find min and max, this locates the smallest and largest value in the binary tree
